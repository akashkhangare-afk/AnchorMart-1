# Legacy — archived split-file build

These files are an **older, multi-file version** of the AnchorMart admin UI
(`css/` stylesheets + `js/app.js` logic) that paired with a now-removed
`index.html`.

**They are not used by the application.** The live app is the self-contained
`../index.html`, which has its own inline CSS and JavaScript. Editing anything
here has no effect on the running app.

Kept only for historical reference. Safe to delete if no longer needed.
